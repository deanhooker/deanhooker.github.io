Kit Bitz

https://arcane-sea-80742.herokuapp.com/login

An inventory management website for electronics hobbists to track inventory and share kits.

After creating an account, the user will be able to search for electronics components using Octopart's API and add components to their inventory. From the inventory tab the user is able to manage their current inventory, whether updating quantity on hand or removing the components entirely.

Study Buddy

https://tannerwwhite.github.io/study_buddy/

Study Buddy is a study session planning app designed for students to create meetups. The app then uses API's to provide the location on Google Maps as well as a 4-day weather forecast at that location.

Data for this app is stored in Firebase. SASS was used for styling.

Friend Finder

https://still-eyrie-65194.herokuapp.com

Friend Finder is a Full-Stack "dating app" designed to match the user up to a previous user based on survey answers. The initial seed data was popular movie characters.

This app was created using Bootsrap on the front end and Express.js on the back end.

Bamazon

https://github.com/deanhooker/bamazon

Bamazon is an Amazon-like CLI storefront split into two views. The customer view allows the customer to "purchase" items from the store, provided there is sufficient stock, and subracts the corresponding inventory. The manager view allows the manager to view and edit inventory.

Bamazon was created using Node.js, an SQL database and the inquirer NPM package.
